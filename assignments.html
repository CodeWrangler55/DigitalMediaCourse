Assignments

Major Projects and Deliverables

Assignments are built around repeated practice: brief, research, draft, critique, revision, final export, and reflection.

Project 1

Poster Redesign

Redesign a weak event poster using hierarchy, contrast, alignment, typographic scale, and a clear call to action.

  • Deliverables: original analysis, 3 thumbnails, draft, final poster, 150-word reflection.
  • Skills: composition, typography, visual hierarchy, critique.

Project 2

Image Composite Campaign Visual

Create a polished composite image that supports a campaign message while demonstrating clean masking, correction, and export.

  • Deliverables: source log, layered working file, final image, process screenshots.
  • Skills: raster editing, compositing, ethics, file management.

Project 3

Logo and Mini Brand Guide

Develop a logo system and concise brand guide for a fictional or local organization.

  • Deliverables: sketches, vector logo, color/type system, usage examples.
  • Skills: vector construction, brand consistency, design rationale.

Project 4

Campaign Collateral Set

Extend the identity into print and digital assets that work together as a campaign.

  • Deliverables: print layout, social set, web landing concept, export package.
  • Skills: layout, multi-format production, responsive thinking.

Project 5

Short Promotional Video

Produce a 15-30 second promotional video or motion social post for the campaign.

  • Deliverables: storyboard, shot list or asset list, rough cut, final video, captions.
  • Skills: story, pacing, titles, sound, compression, accessibility.

Final

Brand Campaign Case Study

Package the semester work into a complete campaign case study with final assets, process documentation, critique response, and reflection.

  • Deliverables: final visual system, portfolio page, presentation, reflection.
  • Skills: professional presentation, revision, portfolio storytelling.
