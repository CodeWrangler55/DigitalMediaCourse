Syllabus

Applied Graphic Design for Digital Media

A studio course in visual communication, image editing, typography, layout, branding, web visuals, and short promotional video for digital media students.

Course Description

This course introduces students to professional graphic design practice across print and digital media. Students learn to analyze visual messages, build design systems, prepare files for different outputs, critique work constructively, and present a portfolio-ready brand campaign.

Course Format

  • 16-week studio course
  • Weekly demonstrations and applied lab work
  • Critique and revision built into major projects
  • Final public-facing campaign case study

Learning Outcomes

Design principles

Apply hierarchy, contrast, alignment, proximity, repetition, balance, and composition to solve communication problems.

Digital production

Create raster, vector, layout, web, and short video assets using industry tools and accessible alternatives.

Process

Use research, sketches, critique, iteration, and documentation to improve design decisions.

Professional presentation

Package final work as a coherent campaign system with a clear written case study.

Tools

Students may use Adobe Creative Cloud when available. Public-access alternatives are included so skill development stays focused on design outcomes rather than a single software license.

  • Image editing: Photoshop, Photopea, GIMP, Krita
  • Vector design: Illustrator, Inkscape, Figma
  • Layout: InDesign, Canva, Affinity Publisher, Scribus
  • Video: Premiere Pro, DaVinci Resolve, CapCut

Teaching Philosophy in Practice

The course assumes students arrive with different software access, confidence levels, and career goals. Assignments are scaffolded, critique is structured, and every major skill leads to a portfolio artifact students can explain.
